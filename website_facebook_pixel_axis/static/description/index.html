Facebook Pixel Integration in odoo | Facebook Pixel for odoo website
Facebook Pixel Track Odoo Apps

Facebook Pixel is an analytical tool which provides collecting data about various actions taken by the visitors in Odoo website. Facebook Pixel is a piece of JavaScript code for your website that enables you to measure, optimize, and build audiences for your ad campaigns. Using the Facebook Pixel, you can leverage the actions people take on your odoo website across devices to inform more.

This module is supported in odoo community and enterprise edition

Key Feature of Facebook Pixel in odoo

Easily Configurable in odoo

Easily configure Facebook Pixel in odoo website.

Add Facebook pixel in odoo

Using this option Facebook Pixel can be added to any odoo website

Track Analytics

Track & check analytics in Facebook analytics

Facebook Tracks

Deploy Facebook tracks events.

Enable/Disable

Easy way to enable or disable Facebook Pixel

TrackingID Add

Add Tracking ID and copy paste in odoo website backend, facebook pixel is start

Tracking Activity

Complete Registration, Initiate Checkout, Purchase, Search, and View Content events - track in odoo

Follow below steps for setup this Facebook Pixel in Odoo website:

Click on Pixel and copy - then add to odoo configuration backend

Go to Website > Configuration > Settings > Website > Facebook Pixel

The Facebook Pixel will be activated on all pages of the website..
